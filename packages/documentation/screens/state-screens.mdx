전체 화면을 차지하는 상태 표시 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Screens API Reference](/api-reference/screens)를 참조하세요.
</Note>

## tc-loading-screen

로딩 중인 상태를 표시하는 전체 화면 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-loading-screen message="데이터를 불러오는 중..."></tc-loading-screen>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/screens';

html`
  <tc-loading-screen
    message="사용자 정보를 불러오는 중..."
    spinner-size="lg"
  ></tc-loading-screen>
`;
```
</Tab>
</Tabs>

## tc-error-screen

에러 상태를 표시하는 전체 화면 컴포넌트입니다.

**이벤트**: `tc-retry`

<Tabs>
<Tab title="HTML">
```html
<tc-error-screen
  title="연결 실패"
  message="서버 연결에 실패했습니다."
  retry-label="다시 시도"
></tc-error-screen>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/screens';

html`
  <tc-error-screen
    title="연결 실패"
    message="서버 연결에 실패했습니다."
    retry-label="다시 시도"
    @tc-retry=${handleRetry}
  ></tc-error-screen>
`;
```
</Tab>
</Tabs>

## tc-success-screen

성공 상태를 표시하는 전체 화면 컴포넌트입니다.

**이벤트**: `tc-action`
**메서드**: `playAnimation()`

<Tabs>
<Tab title="HTML">
```html
<tc-success-screen
  title="결제 완료!"
  message="주문이 성공적으로 처리되었습니다."
  action-label="계속 쇼핑하기"
></tc-success-screen>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/screens';

html`
  <tc-success-screen
    title="결제 완료!"
    message="주문이 성공적으로 처리되었습니다."
    action-label="계속 쇼핑하기"
    @tc-action=${handleAction}
  ></tc-success-screen>
`;
```
</Tab>
</Tabs>

## 통합 예제: 데이터 로딩 플로우

```typescript
type ScreenState = 'loading' | 'error' | 'success' | 'content';

function renderScreen(state: ScreenState) {
  switch (state) {
    case 'loading':
      return html`<tc-loading-screen message="처리 중..."></tc-loading-screen>`;
    case 'error':
      return html`<tc-error-screen @tc-retry=${retry}></tc-error-screen>`;
    case 'success':
      return html`<tc-success-screen title="완료" @tc-action=${next}></tc-success-screen>`;
    default:
      return html`<!-- 메인 콘텐츠 -->`;
  }
}
```

## CSS 커스터마이징

CSS Parts를 사용하여 개별 스타일링 가능:

```css
tc-error-screen::part(icon) {
  background: var(--error-bg);
}

tc-success-screen::part(title) {
  font-size: 1.5rem;
  font-weight: 700;
}
```
