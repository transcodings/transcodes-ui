## 아키텍처

Transcodes UI는 4계층 구조로 설계되었습니다:

```
┌─────────────────────────────────┐
│         Screens                 │  전체 화면 레이아웃
├─────────────────────────────────┤
│         Widgets                 │  복합 컴포넌트
├─────────────────────────────────┤
│        Primitives               │  기본 UI 요소
├─────────────────────────────────┤
│      Design Tokens              │  디자인 시스템
└─────────────────────────────────┘
```

- **Design Tokens**: 색상, 타이포그래피, 간격 등 디자인 시스템
- **Primitives**: Button, Input, Card 등 기본 UI 요소 (21개)
- **Widgets**: Modal, Banner 등 복합 컴포넌트 (8개)
- **Screens**: Loading, Error 등 전체 화면 레이아웃 (3개)
