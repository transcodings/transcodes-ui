## Setup

### 1. CSS 토큰 임포트

디자인 토큰 CSS를 자동으로 로드합니다:

```typescript
// main.ts or index.ts
import '@transcodes/design-tokens';
```

이 한 줄이 다음을 자동으로 포함합니다:
- 기본 색상, 타이포그래피, 간격 토큰
- 다크 모드 토큰
- 컴포넌트 스타일 (CSS reset 포함)

### 2. 컴포넌트 임포트

## Usage

### Vanilla JavaScript

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>Transcodes UI</title>
</head>
<body>
  <tc-button variant="primary">Click me</tc-button>
  <tc-input placeholder="Enter text"></tc-input>

  <script type="module">
    import '@transcodes/design-tokens';
    import '@transcodes/ui-components/primitives';

    const button = document.querySelector('tc-button');
    button.addEventListener('tc-click', (e) => {
      console.log('Button clicked!', e.detail);
    });
  </script>
</body>
</html>
```

### React

```tsx
import '@transcodes/design-tokens';
import '@transcodes/ui-components/primitives';

function App() {
  const handleClick = (e: CustomEvent) => {
    console.log('Clicked!', e.detail);
  };

  return (
    <div>
      <tc-button
        variant="primary"
        onTcClick={handleClick}
      >
        Click me
      </tc-button>
      <tc-input placeholder="Enter text" />
    </div>
  );
}

export default App;
```

<Note>
React에서 커스텀 이벤트를 사용하려면 `onTcClick`처럼 camelCase로 작성하거나, `ref`를 사용하여 `addEventListener`로 등록하세요.
</Note>

### Lit

```typescript
import { LitElement, html } from 'lit';
import { customElement } from 'lit/decorators.js';
import '@transcodes/design-tokens';
import '@transcodes/ui-components/primitives';

@customElement('my-app')
export class MyApp extends LitElement {
  render() {
    return html\`
      <tc-button variant="primary" @tc-click=\${this.handleClick}>
        Click me
      </tc-button>
      <tc-input placeholder="Enter text"></tc-input>
    \`;
  }

  private handleClick(e: CustomEvent) {
    console.log('Clicked!', e.detail);
  }
}
```

### TypeScript

```typescript
import '@transcodes/design-tokens';
import '@transcodes/ui-components/primitives';

// 타입 지원 (선택 사항)
import type { TcButton } from '@transcodes/ui-components/primitives/tc-button.js';

const button = document.querySelector<TcButton>('tc-button');
if (button) {
  button.variant = 'secondary';
  button.disabled = true;
}
```

## 다크 모드

다크 모드는 `data-theme` 속성으로 제어합니다:

```javascript
// 다크 모드 활성화
document.documentElement.setAttribute('data-theme', 'dark');

// 라이트 모드로 전환
document.documentElement.setAttribute('data-theme', 'light');

// 토글
const isDark = document.documentElement.dataset.theme === 'dark';
document.documentElement.setAttribute('data-theme', isDark ? 'light' : 'dark');
```

## Tree-shaking

필요한 컴포넌트만 임포트하여 번들 크기를 최소화하세요:

```typescript
// ❌ 252KB - 모든 컴포넌트
import '@transcodes/ui-components';

// ✅ ~50KB - Primitives만
import '@transcodes/ui-components/primitives';

// ✅ ~4KB - 개별 컴포넌트
import '@transcodes/ui-components/primitives/tc-button.js';
import '@transcodes/ui-components/primitives/tc-input.js';
```

<Info>
번들러(Vite, Webpack)가 자동으로 tree-shaking을 수행합니다. 별도 설정 불필요!
</Info>
