화면 배경에 장식용 floating blob을 렌더링하는 컴포넌트입니다. 페이지의 시각적 생동감을 더하면서 콘텐츠 위에 나타나지 않도록 설계되었습니다.

## tc-page-decoration

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `variant` | `'primary' \| 'success' \| 'error'` | `'primary'` | 색상 변형 |

### CSS Parts

| Part 이름 | 설명 |
|----------|------|
| `decoration` | 장식 컨테이너 |
| `blob` | 개별 blob 요소 |

### Variants

#### Primary (기본)
자주색 그래디언트의 3개 floating blob입니다.

```html
<tc-page-decoration variant="primary"></tc-page-decoration>
```

#### Success
초록색 그래디언트의 3개 floating blob입니다.

```html
<tc-page-decoration variant="success"></tc-page-decoration>
```

#### Error
빨간색 그래디언트의 3개 floating blob입니다.

```html
<tc-page-decoration variant="error"></tc-page-decoration>
```

### 레이아웃 구조

Page Decoration은 고정 위치에 있으므로, 페이지 구조를 올바르게 설정해야 합니다:

```html
<html>
  <body>
    <!-- 장식 (z-index: 0) -->
    <tc-page-decoration></tc-page-decoration>

    <!-- 메인 콘텐츠 (z-index: 10 이상) -->
    <main style="position: relative; z-index: 10;">
      <header>...</header>
      <section>...</section>
      <footer>...</footer>
    </main>
  </body>
</html>
```

<Note>
장식이 콘텐츠 위에 나타나는 것을 방지하려면 메인 콘텐츠에 `position: relative`와 `z-index: 10` 이상을 설정하세요.
</Note>

### 커스텀 스타일

```css
/* blob 크기 조정 */
tc-page-decoration::part(blob) {
  opacity: 0.5; /* 더 투명하게 */
  filter: blur(10px); /* 더 부드럽게 */
}

/* 특정 blob만 조정 */
tc-page-decoration::part(blob):first-of-type {
  opacity: 0.3;
}
```

### 다양한 사용 사례

#### 1. 로딩 페이지

```html

<main style="display: flex; align-items: center; justify-content: center; height: 100vh;">
  <tc-spinner size="3rem"></tc-spinner>
  <p>로딩 중...</p>
</main>
```

#### 2. 성공 페이지

```html
<tc-page-decoration variant="success"></tc-page-decoration>

<main style="display: flex; flex-direction: column; align-items: center; justify-content: center; height: 100vh;">
  <tc-icon name="check-circle" size="4rem" color="var(--accent-success)"></tc-icon>
  <h1>완료되었습니다!</h1>
  <p>요청이 성공적으로 처리되었습니다</p>

```

#### 3. 에러 페이지


### 애니메이션 상세

Page Decoration의 3개 blob은 다양한 속도와 방향으로 애니메이션됩니다:

```css
/* blob-1: 12초, 정상 방향 */
.blob-1 {
  animation: decorFloat 12s ease-in-out infinite;
}

/* blob-2: 10초, 역방향 */
.blob-2 {
  animation: decorFloat 10s ease-in-out infinite reverse;
}

/* blob-3: 14초, 2초 지연 */
.blob-3 {
  animation: decorFloat 14s ease-in-out infinite 2s;
}

@keyframes decorFloat {
  0%, 100% {
    transform: translate(0, 0) scale(1);
  }
  33% {
    transform: translate(6px, -8px) scale(1.08);
  }
  66% {
    transform: translate(-4px, -4px) scale(1.15);
  }
}
```

### 모션 줄이기

사용자가 `prefers-reduced-motion`을 활성화한 경우 자동으로 애니메이션이 비활성화됩니다:

```css
@media (prefers-reduced-motion: reduce) {
  tc-page-decoration::part(blob) {
    animation: none;
  }
}
```

### 색상 커스터마이징

Design Tokens를 통해 blob 색상이 결정됩니다:

```html
<!-- 라이트 모드 (기본) -->
<html data-theme="light">
  <tc-page-decoration variant="primary"></tc-page-decoration>
</html>

<!-- 다크 모드 -->
<html data-theme="dark">
  <tc-page-decoration variant="primary"></tc-page-decoration>
</html>
```

다크 모드에서도 자동으로 적절한 색상이 적용됩니다.

### 반응형 디자인

```javascript
// 모바일과 데스크톱에서 다른 변형 사용
function setDecorationVariant() {
  const decoration = document.querySelector('tc-page-decoration');
  const isMobile = window.innerWidth < 768;

  // 모바일에서는 더 경량의 장식
  decoration.variant = isMobile ? 'primary' : 'primary';
}

window.addEventListener('resize', setDecorationVariant);
setDecorationVariant();
```

### 동적 변형 변경

```javascript
// 페이지 상태에 따라 변형 변경
const decoration = document.querySelector('tc-page-decoration');

// 로딩 중
decoration.variant = 'primary';

// 성공
setTimeout(() => {
  decoration.variant = 'success';
}, 2000);

// 에러 (옵션)
// decoration.variant = 'error';
```

### 성능 팁

1. **조기 렌더링**: Page Decoration은 최상단에 배치하여 조기에 렌더링되도록 하세요.

```html
<html>
  <head>
    <style>
      tc-page-decoration {
        /* 빠른 페인트 */
        will-change: transform;
      }
    </style>
  </head>
  <body>
    <tc-page-decoration></tc-page-decoration>
    <!-- 나머지 콘텐츠 -->
  </body>
</html>
```

2. **포인터 이벤트 무시**: `pointer-events: none`이 자동으로 설정되어 있으므로 성능에 영향을 주지 않습니다.

3. **GPU 가속**: Transform 애니메이션을 사용하므로 GPU 가속 혜택을 받습니다.

## 주요 특징

- **고정 배경**: `position: fixed`로 설정되어 스크롤 시에도 유지
- **비상호작용**: `pointer-events: none`으로 설정되어 콘텐츠 접근성 영향 없음
- **흐림 효과**: 장식의 경계를 부드럽게 처리하여 시각적 부드러움
- **동적 blob**: 3개의 blob이 다양한 속도로 움직임
- **색상 변형**: Primary, Success, Error 3가지 변형 제공
- **다크 모드 지원**: 자동으로 다크 모드 색상 적용
- **모션 감지**: `prefers-reduced-motion` 존중
