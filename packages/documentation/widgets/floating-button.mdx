# 플로팅 버튼 (Floating Button)

화면의 고정된 위치에 표시되는 플로팅 액션 버튼(FAB) 컴포넌트입니다. 4가지 위치를 지원합니다.

## tc-floating-button

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `position` | `'bottom-right' \| 'bottom-left' \| 'top-right' \| 'top-left'` | `'bottom-right'` | 버튼 위치 |
| `size` | `string` | `'3.5rem'` | 버튼 크기 (CSS 값) |
| `disabled` | `boolean` | `false` | 버튼 비활성화 |
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Events

| 이벤트명 | 설명 |
|---------|------|
| `tc-click` | 버튼 클릭 시 발생 |

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 버튼 콘텐츠 (아이콘 권장) |

### CSS Parts

| Part 이름 | 설명 |
|----------|------|
| `button` | 버튼 요소 |

### 위치 기준

- **bottom-right**: 하단 우측 (기본값)
- **bottom-left**: 하단 좌측
- **top-right**: 상단 우측
- **top-left**: 상단 좌측

모든 위치는 `var(--space-lg)` 간격으로 설정됩니다.

### 커스텀 스타일

```html
<!-- sx prop을 통한 커스텀 스타일 -->
<tc-floating-button
  sx='{
    "background": "linear-gradient(135deg, #667eea 0%, #764ba2 100%)",
    "box-shadow": "0 8px 24px rgba(102, 126, 234, 0.4)"
  }'
>
  <tc-icon name="star"></tc-icon>
</tc-floating-button>
```

### CSS로 커스터마이징

```css
/* 모든 FAB 스타일 변경 */
tc-floating-button::part(button) {
  border-radius: 8px; /* 정사각형으로 변경 */
  font-weight: bold;
}

/* 특정 FAB만 스타일 변경 */
#customFab::part(button) {
  width: 5rem;
  height: 5rem;
  background: linear-gradient(135deg, #667eea, #764ba2);
}

/* 호버 상태 */
tc-floating-button::part(button):hover {
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.3);
}
```

### 다중 FAB 배치

화면에 여러 FAB를 배치할 수 있습니다:

```html
<!-- 우측 FAB 3개 -->
<tc-floating-button position="bottom-right" sx='{"bottom": "1rem"}'>
  <tc-icon name="message"></tc-icon>
</tc-floating-button>

<tc-floating-button position="bottom-right" sx='{"bottom": "5rem"}'>
  <tc-icon name="phone"></tc-icon>
</tc-floating-button>

<tc-floating-button position="bottom-right" sx='{"bottom": "9rem"}'>
  <tc-icon name="mail"></tc-icon>
</tc-floating-button>
```

### 애니메이션 FAB

```javascript
// 펄스 애니메이션
const fab = document.querySelector('tc-floating-button');

// CSS 애니메이션 추가
fab.style.animation = 'pulse 2s infinite';

// 글로벌 스타일
const style = document.createElement('style');
style.textContent = `
  @keyframes pulse {
    0%, 100% {
      box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
    }
    50% {
      box-shadow: 0 4px 24px rgba(102, 126, 234, 0.8);
    }
  }
`;
document.head.appendChild(style);
```

### 접근성

버튼에는 자동으로 `aria-label="Floating action"`이 설정됩니다. 더 명확한 레이블을 위해 title 속성을 추가할 수 있습니다:

```html
<tc-floating-button title="메시지 열기">
  <tc-icon name="message"></tc-icon>
</tc-floating-button>
```

### 반응형 위치

```javascript
// 모바일/데스크톱에서 다른 위치 사용
function setFabPosition() {
  const fab = document.querySelector('tc-floating-button');
  const isMobile = window.innerWidth < 768;

  fab.position = isMobile ? 'bottom-right' : 'bottom-left';
}

window.addEventListener('resize', setFabPosition);
setFabPosition();
```

### 실전 예제

```javascript
// 스크롤 위치에 따른 FAB 표시/숨김
const fab = document.querySelector('tc-floating-button');

window.addEventListener('scroll', () => {
  if (window.scrollY > 300) {
    fab.style.opacity = '1';
    fab.style.pointerEvents = 'auto';
  } else {
    fab.style.opacity = '0';
    fab.style.pointerEvents = 'none';
  }
});

// 클릭 시 상단으로 스크롤
fab.addEventListener('tc-click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});
```

## 주요 특징

- **고정 위치**: `position: fixed`로 뷰포트에 고정
- **Z-index**: 100으로 설정되어 대부분의 콘텐츠 위에 표시
- **포인터 이벤트**: 클릭 불가능한 영역과 겹쳐도 클릭 가능
- **다크 모드**: 자동으로 다크 모드 지원
- **애니메이션**: 호버 시 부드러운 스케일 애니메이션
- **터치 친화적**: 모바일에서 쉽게 클릭 가능한 크기
