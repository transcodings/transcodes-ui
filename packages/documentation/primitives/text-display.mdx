텍스트 표현과 시각적 요소를 위한 4개의 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-text

동적 태그 렌더링을 지원하는 텍스트 컴포넌트입니다.

**색상 옵션**: `primary` | `secondary` | `tertiary` | `success` | `error` | `warning` | `info`

<Tabs>
<Tab title="HTML">
```html
<tc-text>기본 텍스트</tc-text>
<tc-text tag="h1" size="2xl" weight="700">페이지 제목</tc-text>
<tc-text size="sm" color="secondary">보조 텍스트</tc-text>
<tc-text color="error">에러 메시지</tc-text>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-text.js';

html`
  <tc-text tag="h1" size="2xl" weight="700">제목</tc-text>
  <tc-text color="secondary">설명</tc-text>
`;
```
</Tab>
</Tabs>

## tc-icon

39개의 내장 아이콘을 지원하는 아이콘 컴포넌트입니다.

**아이콘 목록**:
- **네비게이션**: `arrow-left`, `arrow-right`, `chevron-right`, `chevron-left`
- **상태**: `check`, `x`, `close`, `error`, `alert-circle`, `info`, `warning`, `success`
- **인증**: `biometric`, `email`, `passkey`, `totp`, `email-otp`, `qrcode`, `key`
- **브랜드**: `apple`, `google`, `windows`, `samsung`
- **기타**: `phone`, `lock`, `person`, `device`, `bell`, `download`, `wifi-off`, `loading`, `loader`

<Tabs>
<Tab title="HTML">
```html
<tc-icon name="info"></tc-icon>
<tc-icon name="check" .sx=${{ '--icon-size': '2rem', '--icon-color': 'green' }}></tc-icon>
<tc-icon name="biometric"></tc-icon>
<tc-icon name="apple"></tc-icon>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-icon.js';

html`
  <tc-icon name="check" .sx=${{ '--icon-size': '2rem' }}></tc-icon>
`;
```
</Tab>
</Tabs>

## tc-symbol

원형 심볼/아바타 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-symbol>AB</tc-symbol>
<tc-symbol><tc-icon name="person"></tc-icon></tc-symbol>
<tc-symbol .sx=${{ '--symbol-size': '4rem', '--symbol-bg': '#FF6B6B', '--symbol-color': 'white' }}>E</tc-symbol>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-symbol.js';

html`
  <tc-symbol .sx=${{ '--symbol-bg': 'var(--accent-success)' }}>
    <tc-icon name="check"></tc-icon>
  </tc-symbol>
`;
```
</Tab>
</Tabs>
