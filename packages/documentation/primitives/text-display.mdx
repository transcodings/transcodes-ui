# 텍스트 & 디스플레이 (Text & Display)

텍스트 표현과 시각적 요소를 위한 4개의 컴포넌트입니다.

## tc-text

동적 태그 렌더링을 지원하는 텍스트 컴포넌트입니다. 시맨틱 HTML과 일관된 스타일링을 제공합니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `tag` | `'p' \| 'span' \| 'div' \| 'h1' \| ... \| 'h6' \| 'label'` | `'p'` | 렌더링 HTML 태그 |
| `size` | `'sm' \| 'base' \| 'lg' \| 'xl' \| '2xl'` | - | 텍스트 크기 |
| `weight` | `'400' \| '500' \| '600' \| '700'` | - | 폰트 무게 |
| `color` | `string` | - | 텍스트 색상 (시맨틱 이름 또는 CSS 값) |
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Color Options

시맨틱 색상: `'primary' \| 'secondary' \| 'tertiary' \| 'muted' \| 'accent' \| 'success' \| 'error' \| 'warning' \| 'info'`

또는 CSS 색상: `'#FF0000'`, `'rgb(255, 0, 0)'`, `'var(--ink-black)'`

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 텍스트 콘텐츠 |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 기본 텍스트 -->
<tc-text>기본 텍스트</tc-text>

<!-- 제목 -->
<tc-text tag="h1" size="2xl" weight="700">
  페이지 제목
</tc-text>

<!-- 부제 -->
<tc-text tag="h2" size="lg" weight="600">
  부제목
</tc-text>

<!-- 작은 텍스트 -->
<tc-text size="sm" color="muted">
  보조 텍스트
</tc-text>

<!-- 강조 텍스트 -->
<tc-text color="accent" weight="600">
  강조된 텍스트
</tc-text>

<!-- 에러 메시지 -->
<tc-text color="error" size="sm">
  오류가 발생했습니다
</tc-text>

<!-- 성공 메시지 -->
<tc-text color="success" weight="500">
  작업이 완료되었습니다
</tc-text>

<!-- 라벨 -->
<tc-text tag="label" weight="500">
  사용자명
</tc-text>
```
</Tab>

## tc-icon

내장 아이콘 세트를 가진 아이콘 컴포넌트입니다. 39개의 아이콘을 지원합니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `name` | `IconName` | `'info'` | 아이콘 이름 |
| `size` | `string` | `'1.5rem'` | 아이콘 크기 |
| `color` | `string` | `'currentColor'` | 아이콘 색상 |

### Supported Icons

**네비게이션**
- `arrow-left`, `arrow-right`, `chevron-right`, `chevron-left`

**상태**
- `check`, `x`, `close`, `error`, `alert-circle`, `info`, `warning`, `success`

**로딩**
- `loading`, `loader`

**인증**
- `biometric`, `email`, `passkey`, `totp`, `email-otp`, `qrcode`, `key`

**브랜드**
- `apple`, `google`, `windows`, `samsung`

**기타**
- `phone`, `lock`, `person`, `device`, `bell`, `download`, `wifi-off`

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 기본 아이콘 -->
<tc-icon name="info"></tc-icon>

<!-- 다양한 크기 -->
<tc-icon name="check" size="1rem"></tc-icon>
<tc-icon name="check" size="2rem"></tc-icon>
<tc-icon name="check" size="3rem"></tc-icon>

<!-- 커스텀 색상 -->
<tc-icon name="success" color="#00AA00"></tc-icon>
<tc-icon name="error" color="var(--error-base)"></tc-icon>
<tc-icon name="warning" color="rgb(255, 165, 0)"></tc-icon>

<!-- 네비게이션 아이콘 -->
<tc-icon name="arrow-left"></tc-icon>
<tc-icon name="chevron-right"></tc-icon>

<!-- 인증 아이콘 -->
<tc-icon name="biometric"></tc-icon>
<tc-icon name="email"></tc-icon>
<tc-icon name="passkey"></tc-icon>

<!-- 브랜드 아이콘 -->
<tc-icon name="apple"></tc-icon>
<tc-icon name="google"></tc-icon>
```
</Tab>

## tc-symbol

원형 심볼 또는 아바타 컴포넌트입니다. 아이콘, 텍스트, 이미지 등을 표시할 수 있습니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `size` | `string` | `'3rem'` | 심볼 크기 |
| `background` | `string` | `'var(--paper-cream)'` | 배경색 |
| `color` | `string` | `'var(--ink-dark)'` | 텍스트/아이콘 색상 |

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 심볼 콘텐츠 (텍스트, 아이콘 등) |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 텍스트 심볼 -->
<tc-symbol>AB</tc-symbol>
<tc-symbol>J</tc-symbol>
<tc-symbol>?</tc-symbol>

<!-- 아이콘 심볼 -->
<tc-symbol>
  <tc-icon name="user"></tc-icon>
</tc-symbol>

<!-- 커스텀 크기 -->
<tc-symbol size="2rem">SM</tc-symbol>
<tc-symbol size="4rem">LG</tc-symbol>

<!-- 커스텀 색상 -->
<tc-symbol background="#FF6B6B" color="white">
  E
</tc-symbol>

<tc-symbol background="#4ECDC4" color="white">
  S
</tc-symbol>

<!-- 상태 색상 -->
<tc-symbol background="var(--accent-success)" color="white">
  <tc-icon name="check"></tc-icon>
</tc-symbol>

<tc-symbol background="var(--error-bg)" color="var(--error-base)">
  <tc-icon name="error"></tc-icon>
</tc-symbol>
```
</Tab>

## tc-divider

구분선 컴포넌트입니다. 텍스트 없이 간단한 선 또는 텍스트와 함께 표시할 수 있습니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `color` | `string` | `'var(--ink-faint)'` | 선의 색상 |
| `spacing` | `string` | `'var(--space-md)'` | 상하 여백 |
| `text` | `string` | `''` | 표시할 텍스트 (빈 문자열이면 단순 선) |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 기본 구분선 -->
<tc-divider></tc-divider>

<!-- 텍스트가 있는 구분선 -->
<tc-divider text="또는"></tc-divider>

<!-- 커스텀 색상 -->
<tc-divider color="var(--accent-primary)"></tc-divider>

<!-- 커스텀 간격 -->
<tc-divider spacing="3rem"></tc-divider>

<!-- 텍스트와 커스텀 스타일 -->
<tc-divider
  text="또는 계속하기"
  color="var(--accent-secondary)"
  spacing="2rem"
></tc-divider>
```
</Tab>

<Tab title="React">
```jsx
import { TcDivider } from '@transcodes/ui-components/primitives';

export function DividerDemo() {
```
