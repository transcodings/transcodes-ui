## tc-card

그림자와 보더 효과가 있는 카드 컴포넌트입니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `no-border` | `boolean` | `false` | 내부 보더 효과 제거 |
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 카드 콘텐츠 |

### Usage

<Tabs>
<Tab title="HTML">
```html
<tc-card>
  <h3>카드 제목</h3>
  <p>카드 콘텐츠입니다.</p>
</tc-card>

<tc-card no-border>
  <h3>보더 없는 카드</h3>
  <p>내부 보더 효과가 없습니다.</p>
</tc-card>

<tc-card sx='{"padding": "2rem", "background": "#f5f5f5"}'>
  <h3>커스텀 스타일 카드</h3>
  <p>sx prop으로 스타일을 오버라이드했습니다.</p>
</tc-card>
```
</Tab>

<Tab title="React">

```jsx
}
```
</Tab>

## tc-box

기본 컨테이너 컴포넌트입니다. `display: contents`를 사용하여 DOM 계층 구조를 변경하지 않습니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 박스 콘텐츠 |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 스타일링이 필요한 래퍼 -->
<tc-box sx='{"display": "flex", "gap": "1rem"}'>
  <div>항목 1</div>
  <div>항목 2</div>
  <div>항목 3</div>
</tc-box>

<!-- 조건부 래핑 -->
<tc-box sx='{"padding": "1rem"}'>
  <h2>제목</h2>
  <p>본문 내용</p>
</tc-box>
```
</Tab>

<Tab title="React">
```jsx
import { TcBox } from '@transcodes/ui-components/primitives';

export function BoxDemo() {
  return (
    <>
      <TcBox sx={{ display: 'flex', gap: '1rem' }}>

      </TcBox>

      <TcBox sx={{ padding: '1rem' }}>
        <h2>제목</h2>
        <p>본문 내용</p>

    </>
  );
}
```
</Tab>

<Tab title="Vue">
```vue

</script>

<template>
  <div>
    <tc-box :sx="{ display: 'flex', gap: '1rem' }">
      <div>항목 1</div>
      <div>항목 2</div>
      <div>항목 3</div>
    </tc-box>

    <tc-box :sx="{ padding: '1rem' }">
      <h2>제목</h2>
      <p>본문 내용</p>
    </tc-box>
```

## tc-container

최대 너비가 제한된 플렉스 컨테이너입니다. 반응형 레이아웃에 최적화되어 있습니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `wide` | `boolean` | `false` | 넓은 최대 너비 적용 |
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Features

- 기본 레이아웃: `flex`, `flex-direction: column`, `align-items: center`
- `--container-max-width` CSS 변수로 최대 너비 제어
- `--container-max-width-wide` CSS 변수로 넓은 너비 제어

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 컨테이너 콘텐츠 |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 기본 컨테이너 -->
<tc-container>
  <h1>페이지 제목</h1>
  <p>중앙 정렬된 콘텐츠</p>
</tc-container>

<!-- 넓은 컨테이너 -->
<tc-container wide>
  <h1>와이드 레이아웃</h1>
  <p>더 넓은 콘텐츠</p>
</tc-container>

<!-- 커스텀 레이아웃 -->
<tc-container sx='{"alignItems": "flex-start", "gap": "2rem"}'>
  <aside>사이드바</aside>
  <main>메인 콘텐츠</main>
</tc-container>
```
</Tab>

<Tab title="React">

## tc-section

세로 배열의 섹션 컨테이너입니다. 일반적으로 페이지의 주요 영역을 구분하는 데 사용됩니다.

### Properties

| 속성명 | 타입 | 기본값 | 설명 |
|-------|------|-------|------|
| `gap` | `string` | `'var(--space-md)'` | 자식 요소 간 간격 |
| `sx` | `Record<string, string \| number>` | `{}` | 인라인 스타일 오버라이드 |

### Features

- 기본 레이아웃: `flex`, `flex-direction: column`
- 모든 너비: `width: 100%`
- 기본 간격: `--space-md` (16px)

### Slots

| 슬롯명 | 설명 |
|-------|------|
| (default) | 섹션 콘텐츠 |

### Usage

<Tabs>
<Tab title="HTML">
```html
<!-- 기본 섹션 -->
<tc-section>
  <h2>섹션 제목</h2>
  <p>섹션 설명</p>
  <button>버튼</button>
</tc-section>

<!-- 간격이 큰 섹션 -->
<tc-section gap="2rem">
  <h2>제목</h2>
  <p>본문</p>
  <footer>푸터</footer>
</tc-section>

<!-- 간격이 작은 섹션 -->
<tc-section gap="0.5rem">
  <h3>항목 1</h3>
  <h3>항목 2</h3>
  <h3>항목 3</h3>
</tc-section>

<!-- 커스텀 스타일 -->
<tc-section sx='{"padding": "2rem", "background": "#f5f5f5"}'>
  <h2>커스텀 섹션</h2>
  <p>배경색과 패딩이 적용된 섹션</p>
</tc-section>
```
</Tab>

## 레이아웃 조합 예시

### 페이지 레이아웃

```jsx
<TcContainer>
  <TcSection gap="3rem">
    <h1>페이지 제목</h1>

    <TcSection gap="2rem">
      <TcCard>
        <h2>섹션 1</h2>
        <p>콘텐츠</p>
      </TcCard>

      <TcCard>
        <h2>섹션 2</h2>
        <p>콘텐츠</p>
      </TcCard>
    </TcSection>
  </TcSection>
</TcContainer>
```

### 양 옆 레이아웃

```jsx
<TcContainer>
  <TcBox sx={{ display: 'flex', gap: '2rem' }}>
    <aside style={{ flex: '0 0 250px' }}>
      <TcSection gap="1rem">
        {/* 사이드바 콘텐츠 */}
      </TcSection>
    </aside>

    <main style={{ flex: 1 }}>
      <TcSection gap="2rem">
        {/* 메인 콘텐츠 */}
      </TcSection>
    </main>
  </TcBox>
</TcContainer>
```

### 그리드 레이아웃

```jsx
<TcContainer>
  <TcBox sx={{
    display: 'grid',
    gridTemplateColumns: 'repeat(auto-fit, minmax(300px, 1fr))',
    gap: '2rem'
  }}>
    <TcCard>...</TcCard>
    <TcCard>...</TcCard>
    <TcCard>...</TcCard>
  </TcBox>
</TcContainer>
```
