콘텐츠를 그룹화하고 레이아웃을 구성하는 4개의 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-card

그림자와 보더 효과가 있는 카드 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-card>
  <h3>카드 제목</h3>
  <p>카드 콘텐츠</p>
</tc-card>

<tc-card no-border>
  <h3>보더 없는 카드</h3>
</tc-card>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-card.js';

html`
  <tc-card .sx=${{ padding: '2rem' }}>
    <h3>카드 제목</h3>
    <p>콘텐츠</p>
  </tc-card>
`;
```
</Tab>
</Tabs>

## tc-box

기본 컨테이너 컴포넌트입니다. `display: contents`를 사용합니다.

<Note>
`display: contents`로 인해 box model 속성(margin, padding, border)이 직접 적용되지 않습니다.
</Note>

<Tabs>
<Tab title="HTML">
```html
<tc-box .sx=${{ display: 'flex', gap: '1rem' }}>
  <div>항목 1</div>
  <div>항목 2</div>
</tc-box>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-box.js';

html`
  <tc-box .sx=${{ display: 'grid', gap: '2rem' }}>
    ${items.map(item => html`<div>${item}</div>`)}
  </tc-box>
`;
```
</Tab>
</Tabs>

## tc-container

최대 너비가 제한된 플렉스 컨테이너입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-container>
  <h1>페이지 제목</h1>
  <p>중앙 정렬 콘텐츠</p>
</tc-container>

<tc-container wide>
  <h1>와이드 레이아웃</h1>
</tc-container>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-container.js';

html`
  <tc-container ?wide=${isWide}>
    <slot></slot>
  </tc-container>
`;
```
</Tab>
</Tabs>

## tc-section

세로 배열의 섹션 컨테이너입니다. 기본 `gap: var(--space-md)`입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-section>
  <h2>섹션 제목</h2>
  <p>섹션 설명</p>
</tc-section>

<tc-section .sx=${{ gap: 'var(--space-lg)' }}>
  <tc-text>항목 1</tc-text>
  <tc-text>항목 2</tc-text>
</tc-section>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-section.js';

html`
  <tc-section .sx=${{ gap: '2rem' }}>
    <tc-text>항목 1</tc-text>
    <tc-text>항목 2</tc-text>
  </tc-section>
`;
```
</Tab>
</Tabs>

## 레이아웃 조합 예시

```typescript
// 페이지 레이아웃
html`
  <tc-container>
    <tc-section .sx=${{ gap: '3rem' }}>
      <tc-text tag="h1" size="2xl">페이지 제목</tc-text>
      <tc-card>섹션 1</tc-card>
      <tc-card>섹션 2</tc-card>
    </tc-section>
  </tc-container>
`;

// 그리드 레이아웃
html`
  <tc-container>
    <tc-box .sx=${{
      display: 'grid',
      gridTemplateColumns: 'repeat(auto-fit, minmax(300px, 1fr))',
      gap: '2rem'
    }}>
      <tc-card>카드 1</tc-card>
      <tc-card>카드 2</tc-card>
      <tc-card>카드 3</tc-card>
    </tc-box>
  </tc-container>
`;
```
