화면 배경에 장식용 floating blob을 렌더링하는 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-page-decoration

**Variants**: `primary` (자주색) | `success` (초록색) | `error` (빨간색)

<Tabs>
<Tab title="HTML">
```html
<tc-page-decoration variant="primary"></tc-page-decoration>
<tc-page-decoration variant="success"></tc-page-decoration>
<tc-page-decoration variant="error"></tc-page-decoration>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-page-decoration.js';

html`
  <tc-page-decoration variant=${pageVariant}></tc-page-decoration>
`;
```
</Tab>
</Tabs>

### 레이아웃 구조

Page Decoration은 `position: absolute`로 배치됩니다. 콘텐츠 위에 나타나지 않도록 메인 콘텐츠에 `z-index`를 설정하세요.

```html
<div style="position: relative; min-height: 100vh;">
  <tc-page-decoration></tc-page-decoration>

  <main style="position: relative; z-index: 10;">
    <h1>메인 콘텐츠</h1>
  </main>
</div>
```

### 사용 사례

```html
<!-- 로딩 페이지 -->
<tc-page-decoration variant="primary"></tc-page-decoration>

<!-- 성공 페이지 -->
<tc-page-decoration variant="success"></tc-page-decoration>

<!-- 에러 페이지 -->
<tc-page-decoration variant="error"></tc-page-decoration>
```

### 주요 특징

- **비상호작용**: `pointer-events: none`
- **Fluid 크기**: Utopia 토큰 사용 (모바일 192px → 데스크톱 352px)
- **모션 감지**: `prefers-reduced-motion` 존중
- **다크 모드**: 자동 색상 적용
