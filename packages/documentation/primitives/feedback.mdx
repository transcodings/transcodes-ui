사용자에게 피드백을 제공하는 4개의 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-toast

자동으로 소멸되는 알림 메시지 컴포넌트입니다.

**메서드**: `show()`, `hide()`

<Tabs>
<Tab title="HTML">
```html
<tc-toast id="toast" variant="success">작업 완료!</tc-toast>
<tc-toast variant="error">오류 발생</tc-toast>
<tc-toast variant="warning" duration="0">경고 (수동 닫기)</tc-toast>

<script>
  const toast = document.getElementById('toast');
  await toast.show();
</script>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-toast.js';

html`
  <tc-toast
    variant="success"
    @tc-dismiss=${handleDismiss}
  >
    작업이 완료되었습니다!
  </tc-toast>
`;
```
</Tab>
</Tabs>

## tc-callout

알림, 공지, 경고 등을 표시하는 박스 컴포넌트입니다.

**슬롯**: `icon` | `default`

<Tabs>
<Tab title="HTML">
```html
<tc-callout variant="info">
  <tc-icon slot="icon" name="info"></tc-icon>
  정보 메시지입니다.
</tc-callout>

<tc-callout variant="success">
  <tc-icon slot="icon" name="check"></tc-icon>
  성공적으로 완료되었습니다.
</tc-callout>

<tc-callout variant="error">
  <tc-icon slot="icon" name="error"></tc-icon>
  오류가 발생했습니다.
</tc-callout>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-callout.js';

html`
  <tc-callout variant="warning">
    <tc-icon slot="icon" name="alert-circle"></tc-icon>
    주의가 필요합니다.
  </tc-callout>
`;
```
</Tab>
</Tabs>

## tc-spinner

로딩 상태를 나타내는 스피너 컴포넌트입니다.

**크기**: `sm` | `md` | `lg` | `auto` (반응형)

<Tabs>
<Tab title="HTML">
```html
<tc-spinner></tc-spinner>
<tc-spinner size="sm"></tc-spinner>
<tc-spinner size="lg"></tc-spinner>
<tc-spinner .sx=${{ '--spinner-color': 'var(--accent-success)' }}></tc-spinner>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-spinner.js';

html`
  <tc-spinner size="md" .sx=${{ '--spinner-color': '#FF6B6B' }}></tc-spinner>
`;
```
</Tab>
</Tabs>

  인터랙티브 예제

## tc-error-message

에러, 경고, 정보 메시지를 표시하는 간편 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-error-message variant="error" message="사용자명은 3자 이상이어야 합니다"></tc-error-message>
<tc-error-message variant="warning" message="비밀번호는 8자 이상이어야 합니다"></tc-error-message>
<tc-error-message variant="info" message="입력 후 엔터를 누르세요"></tc-error-message>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-error-message.js';

html`
  <tc-error-message
    variant="error"
    message=${errorMessage}
  ></tc-error-message>
`;
```
</Tab>
</Tabs>
