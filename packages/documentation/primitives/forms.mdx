폼 구성 및 항목 표시에 사용되는 4개의 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-form-header

폼의 제목, 부제, 공지를 표시하는 헤더 컴포넌트입니다.

**특징**: slideDown/slideUp 애니메이션, 모션 감소 지원

<Tabs>
<Tab title="HTML">
```html
<tc-form-header
  title="회원가입"
  subtitle="새 계정을 만들어주세요"
></tc-form-header>

<tc-form-header
  title="로그인"
  notice="첫 로그인이신가요?"
  no-animation
></tc-form-header>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-form-header.js';

html`
  <tc-form-header
    title="회원가입"
    subtitle="새 계정을 만들어주세요"
  ></tc-form-header>
`;
```
</Tab>
</Tabs>

## tc-chip

라벨, 배지, 태그를 표시하는 칩 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-chip>레이블</tc-chip>
<tc-chip variant="success">완료</tc-chip>
<tc-chip variant="error">실패</tc-chip>
<tc-chip variant="info">새로움</tc-chip>
<tc-chip size="sm">작은 칩</tc-chip>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-chip.js';

html`
  <tc-chip variant="success">
    <tc-icon name="check"></tc-icon>
    완료됨
  </tc-chip>
`;
```
</Tab>
</Tabs>

## tc-item

클릭할 수 없는 항목 컴포넌트입니다.

**슬롯**: `prefix` | `default` | `suffix`

<Tabs>
<Tab title="HTML">
```html
<tc-item>
  <tc-icon slot="prefix" name="user"></tc-icon>
  <tc-text>사용자 프로필</tc-text>
</tc-item>

<tc-item>
  <tc-symbol slot="prefix">AB</tc-symbol>
  <div>
    <tc-text weight="600">Alice Brown</tc-text>
    <tc-text size="sm" color="secondary">Premium Member</tc-text>
  </div>
  <tc-chip slot="suffix" variant="info">5</tc-chip>
</tc-item>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-item.js';

html`
  <tc-item .sx=${{ '--item-gap': 'var(--space-sm)' }}>
    <tc-icon slot="prefix" name="bell"></tc-icon>
    <tc-text>알림</tc-text>
  </tc-item>
`;
```
</Tab>
</Tabs>

## tc-item-button

클릭 가능한 항목 컴포넌트입니다.

**슬롯**: `prefix` | `default` | `suffix`

<Tabs>
<Tab title="HTML">
```html
<tc-item-button>
  <tc-icon slot="prefix" name="person"></tc-icon>
  <tc-text>프로필</tc-text>
</tc-item-button>

<tc-item-button show-arrow="false">
  <tc-text>토글 항목</tc-text>
  <input slot="suffix" type="checkbox" />
</tc-item-button>

<tc-item-button disabled>
  <tc-icon slot="prefix" name="lock"></tc-icon>
  <tc-text>비활성화됨</tc-text>
</tc-item-button>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-item-button.js';

html`
  <tc-item-button @tc-click=${handleClick}>
    <tc-icon slot="prefix" name="person"></tc-icon>
    <tc-text>프로필</tc-text>
  </tc-item-button>
`;
```
</Tab>
</Tabs>

## 완전한 폼 예제

```typescript
html`
  <tc-section .sx=${{ gap: '2rem' }}>
    <tc-form-header
      title="회원가입"
      subtitle="새 계정을 만들어주세요"
    ></tc-form-header>

    <tc-section .sx=${{ gap: '1rem' }}>
      <tc-input label="사용자명" placeholder="username" required></tc-input>
      <tc-input label="이메일" type="email" required></tc-input>
      <tc-input label="비밀번호" type="password" required></tc-input>
    </tc-section>

    <tc-divider text="또는"></tc-divider>

    <tc-button variant="primary">회원가입</tc-button>
  </tc-section>
`;
```
