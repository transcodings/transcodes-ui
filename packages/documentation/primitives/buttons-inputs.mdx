기본적인 사용자 상호작용을 위한 4개의 컴포넌트입니다.

<Note>
모든 Props의 상세 정보는 [Primitives API Reference](/api-reference/primitives)를 참조하세요.
</Note>

## tc-button

상태별 스타일을 지원하는 다목적 버튼 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-button>기본 버튼</tc-button>
<tc-button variant="secondary">세컨더리</tc-button>
<tc-button variant="success">성공</tc-button>
<tc-button disabled>비활성</tc-button>
<tc-button loading>로딩 중...</tc-button>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-button.js';

html`
  <tc-button @tc-click=${handleClick}>클릭</tc-button>
  <tc-button ?loading=${isLoading}>로딩 중...</tc-button>
`;
```
</Tab>
</Tabs>

## tc-input

라벨과 에러 표시가 있는 텍스트 입력 필드입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-input label="사용자명" placeholder="Enter username" required></tc-input>
<tc-input label="이메일" type="email"></tc-input>
<tc-input label="비밀번호" type="password"></tc-input>
<tc-input label="에러 상태" error="이미 사용 중입니다"></tc-input>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-input.js';

html`
  <tc-input
    label="사용자명"
    .value=${username}
    @tc-input=${(e) => username = e.detail.value}
  ></tc-input>
`;
```
</Tab>
</Tabs>

## tc-otp-input

6자리 OTP 입력을 위한 전문화된 컴포넌트입니다.

**특징**:
- 자동 셀 이동 (숫자 입력 시 다음 셀로)
- 백스페이스로 이전 셀 이동
- 화살표 키 네비게이션
- 붙여넣기 지원

<Tabs>
<Tab title="HTML">
```html
<tc-otp-input></tc-otp-input>
<tc-otp-input length="4"></tc-otp-input>
<tc-otp-input error></tc-otp-input>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-otp-input.js';

html`
  <tc-otp-input
    @tc-otp-complete=${(e) => verifyOtp(e.detail.value)}
  ></tc-otp-input>
`;
```
</Tab>
</Tabs>

## tc-input-with-chip

입력 필드 옆에 칩 인디케이터가 표시되는 컴포넌트입니다.

<Tabs>
<Tab title="HTML">
```html
<tc-input-with-chip
  label="이메일"
  placeholder="your@email.com"
  chip-label="이메일"
  chip-icon="email"
  chip-variant="info"
></tc-input-with-chip>

<tc-input-with-chip
  label="계정명"
  chip-label="생체 인증"
  chip-icon="biometric"
  chip-variant="success"
  readonly
></tc-input-with-chip>
```
</Tab>

<Tab title="Lit">
```typescript
import '@transcodes/ui-components/primitives/tc-input-with-chip.js';

html`
  <tc-input-with-chip
    label="이메일"
    chip-label="이메일 인증"
    chip-variant="info"
    @tc-input=${handleInput}
  ></tc-input-with-chip>
`;
```
</Tab>
</Tabs>
