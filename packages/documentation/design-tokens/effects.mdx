## 이펙트 시스템

Transcodes UI의 이펙트 시스템은 **Border Radius**, **Shadows**, **Transitions**, **Animations**로 구성됩니다. 이 토큰들은 컴포넌트의 시각적 품질과 인터랙션 피드백을 담당합니다.

## Border Radius

모서리 둥글기 토큰은 **5개 레벨**로 제공됩니다. `radius-card`는 반응형으로 뷰포트에 따라 조정됩니다.

### 토큰 목록

| 토큰 | 값 | 용도 |
|------|-----|------|
| `radius-sm` | `0.5rem` (8px) | 버튼, 입력 필드 |
| `radius-md` | `0.625rem` (10px) | 작은 카드, 드롭다운 |
| `radius-lg` | `0.75rem` (12px) | 모달, 팝오버 |
| `radius-card` | `clamp(0.75rem, 0.59rem + 0.78vw, 1rem)` | 카드 컴포넌트 (반응형) |
| `radius-full` | `9999px` | 원형 버튼, 아바타, 필(pill) |

### 사용 예시

<CodeGroup>
  <CodeBlock title="CSS">
    ```css
    /* 버튼 */
    .button {
      border-radius: var(--radius-sm);
    }

    /* 카드 - 반응형 둥글기 */
    .card {
      border-radius: var(--radius-card);
    }

    /* 아바타 - 원형 */
    .avatar {
      border-radius: var(--radius-full);
    }

    /* 모달 */
    .modal {
      border-radius: var(--radius-lg);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    const radius = {
      sm: tokens.radiusSm,       // "0.5rem"
      md: tokens.radiusMd,       // "0.625rem"
      lg: tokens.radiusLg,       // "0.75rem"
      card: tokens.radiusCard,   // "clamp(...)"
      full: tokens.radiusFull,   // "9999px"
    };
    ```
  </CodeBlock>
</CodeGroup>

## Shadows

그림자 토큰은 **깊이와 강조**를 표현합니다. 컴포넌트의 시각적 계층과 상호작용 피드백에 사용됩니다.

### 토큰 목록

| 토큰 | 용도 |
|------|------|
| `shadow-overlay` | 드롭다운, 툴팁 |
| `shadow-card` | 카드 컴포넌트 (3레이어) |
| `shadow-button-hover-primary` | Primary 버튼 호버 |
| `shadow-button-hover-success` | Success 버튼 호버 |
| `shadow-input-focus` | 입력 필드 포커스 |
| `shadow-modal` | 모달 다이얼로그 |
| `shadow-focus-ring-width` | 포커스 링 두께 (3px) |

### 상세 값

<CodeGroup>
  <CodeBlock title="shadow-card (3레이어)">
    ```css
    /* 미묘한 상단 하이라이트 + 중간 확산 + 넓은 환경광 */
    box-shadow:
      0 0.0625rem 0.1875rem rgba(26, 26, 26, 0.04),
      0 0.375rem 1rem rgba(26, 26, 26, 0.06),
      0 1.5rem 3rem rgba(26, 26, 26, 0.06);
    ```
  </CodeBlock>
  <CodeBlock title="shadow-button-hover-primary">
    ```css
    /* Primary 색상 기반 글로우 효과 */
    box-shadow:
      0 0.25rem 0.75rem rgba(124, 92, 255, 0.3),
      0 0.125rem 0.25rem rgba(124, 92, 255, 0.2);
    ```
  </CodeBlock>
  <CodeBlock title="shadow-input-focus">
    ```css
    /* 포커스 링 + 내부 인셋 */
    box-shadow:
      0 0 0 0.1875rem rgba(124, 92, 255, 0.12),
      inset 0 0.0625rem 0.125rem rgba(26, 26, 26, 0.04);
    ```
  </CodeBlock>
  <CodeBlock title="shadow-modal">
    ```css
    /* 깊은 모달 그림자 */
    box-shadow: 0 1.5rem 3rem rgba(0, 0, 0, 0.15);
    ```
  </CodeBlock>
</CodeGroup>

### 사용 예시

```css
/* 카드 */
.card {
  box-shadow: var(--shadow-card);
}

/* 버튼 호버 */
.button-primary:hover {
  box-shadow: var(--shadow-button-hover-primary);
}

/* 입력 필드 포커스 */
.input:focus {
  box-shadow: var(--shadow-input-focus);
}

/* 모달 */
.modal {
  box-shadow: var(--shadow-modal);
}

/* 커스텀 포커스 링 */
.custom-focus:focus {
  box-shadow: 0 0 0 var(--shadow-focus-ring-width) var(--alpha-primary20);
}
```

## Transitions

트랜지션 토큰은 **상태 변화의 부드러움**을 제어합니다. 3가지 프리셋과 개별 duration/easing 토큰을 제공합니다.

### 프리셋 트랜지션

| 토큰 | 값 | 용도 |
|------|-----|------|
| `transition-fast` | `150ms ease` | 호버, 포커스 즉각 피드백 |
| `transition-smooth` | `300ms cubic-bezier(0.4, 0, 0.2, 1)` | 일반 상태 변화 |
| `transition-bounce` | `400ms cubic-bezier(0.34, 1.56, 0.64, 1)` | 팝업, 강조 효과 |

### Duration 토큰

개별 duration을 조합해야 할 때 사용합니다:

| 토큰 | 값 | 용도 |
|------|-----|------|
| `duration-instant` | `150ms` | 마이크로 인터랙션 |
| `duration-fast` | `300ms` | 호버/포커스 |
| `duration-smooth` | `400ms` | 모달/오버레이 |
| `duration-slow` | `600ms` | 페이지 애니메이션 |
| `duration-slower` | `800ms` | 복잡한 애니메이션 |

### Easing 토큰

| 토큰 | 값 | 특성 |
|------|-----|------|
| `easing-ease` | `ease` | 기본 easing |
| `easing-ease-in-out` | `cubic-bezier(0.4, 0, 0.2, 1)` | 부드러운 가감속 |
| `easing-bounce` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 바운스 효과 |

### 사용 예시

<CodeGroup>
  <CodeBlock title="프리셋 사용">
    ```css
    /* 빠른 호버 피드백 */
    .button {
      transition: background var(--transition-fast),
                  transform var(--transition-fast);
    }

    /* 부드러운 전환 */
    .modal {
      transition: opacity var(--transition-smooth),
                  transform var(--transition-smooth);
    }

    /* 바운스 효과 */
    .notification {
      transition: transform var(--transition-bounce);
    }
    ```
  </CodeBlock>
  <CodeBlock title="개별 조합">
    ```css
    /* 커스텀 duration + easing 조합 */
    .custom-transition {
      transition-duration: var(--duration-slow);
      transition-timing-function: var(--easing-ease-in-out);
    }

    /* 여러 속성에 다른 타이밍 */
    .complex-transition {
      transition:
        opacity var(--duration-fast) var(--easing-ease),
        transform var(--duration-smooth) var(--easing-bounce);
    }
    ```
  </CodeBlock>
</CodeGroup>

## CSS Animations

Transcodes UI는 **7개의 키프레임 애니메이션**을 제공합니다. `components.css`에 포함되어 있습니다.

### 키프레임 목록

| 애니메이션 | 용도 | 설명 |
|-----------|------|------|
| `fadeInUp` | 페이지/콘텐츠 진입 | 아래에서 위로 페이드인 |
| `slideDown` | 드롭다운, 알림 | 위에서 아래로 슬라이드 |
| `slideUp` | Notice, 토스트 | 아래에서 위로 슬라이드 |
| `spin` | 스피너, 로딩 | 360도 회전 |
| `shake` | 에러 피드백 | 좌우 흔들림 |
| `inkFloat` | 장식 배경 | 부유하는 움직임 |
| `inkDrift` | 장식 배경 | 표류하는 움직임 |

### 키프레임 정의

<CodeGroup>
  <CodeBlock title="fadeInUp">
    ```css
    @keyframes fadeInUp {
      0% {
        opacity: 0;
        transform: translateY(1rem);
      }
      100% {
        opacity: 1;
        transform: translateY(0);
      }
    }
    ```
  </CodeBlock>
  <CodeBlock title="slideDown">
    ```css
    @keyframes slideDown {
      0% {
        opacity: 0;
        transform: translateY(-0.5rem);
      }
      100% {
        opacity: 1;
        transform: translateY(0);
      }
    }
    ```
  </CodeBlock>
  <CodeBlock title="shake">
    ```css
    @keyframes shake {
      0%, 100% {
        transform: translate(0);
      }
      25% {
        transform: translate(-0.25rem);
      }
      75% {
        transform: translate(0.25rem);
      }
    }
    ```
  </CodeBlock>
</CodeGroup>

### Animation Offset 토큰

애니메이션 거리 조정을 위한 토큰:

| 토큰 | 값 | 용도 |
|------|-----|------|
| `offset-slide-up-sm` | `0.25rem` (4px) | 에러 메시지 |
| `offset-slide-up-md` | `0.625rem` (10px) | 화면 진입 |

### 사용 예시

```css
/* 페이지 진입 애니메이션 */
.page-container {
  animation: fadeInUp 0.6s var(--transition-smooth) both;
}

/* 에러 메시지 진입 */
.error-message {
  animation: slideDown 0.3s var(--transition-smooth) both;
}

/* 에러 시 흔들림 */
.error-message.shake {
  animation: shake 0.4s var(--transition-smooth);
}

/* 스피너 */
.spinner {
  animation: spin 0.6s linear infinite;
}

/* 지연 애니메이션 */
.form-title {
  animation: slideDown 0.4s var(--transition-smooth) 0.2s both;
}
```

## Ink Effect (Ripple)

버튼의 **잉크 퍼짐 효과**를 위한 전용 토큰입니다. Material Design의 ripple 효과를 구현합니다.

### Ink Effect 토큰

| 토큰 | 값 | 설명 |
|------|-----|------|
| `ink-effect-ripple-duration` | `600ms` | 리플 확산 시간 |
| `ink-effect-ripple-easing` | `cubic-bezier(0.4, 0, 0.2, 1)` | 리플 easing |
| `ink-effect-spread-scale` | `2.5` | 리플 최대 크기 배율 |
| `ink-effect-fade-duration` | `300ms` | 페이드아웃 시간 |

### 구현 예시

```css
/* 버튼 잉크 효과 */
.button-ink {
  position: relative;
  overflow: hidden;
}

.button-ink::after {
  content: '';
  position: absolute;
  top: 50%;
  left: 50%;
  width: 350px;
  height: 350px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(255,255,255,0.5) 0%, transparent 70%);
  transform: translate(-50%, -50%) scale(0);
  opacity: 0;
  pointer-events: none;
  transition:
    transform var(--ink-effect-ripple-duration) var(--ink-effect-ripple-easing),
    opacity var(--ink-effect-fade-duration) ease;
}

.button-ink:hover::after {
  transform: translate(-50%, -50%) scale(1);
  opacity: 1;
}

.button-ink:active::after {
  transform: translate(-50%, -50%) scale(0.8);
  opacity: 0.8;
}
```

## Ink Spreading Effects

**장식용 배경 애니메이션**입니다. `page-decoration` 컴포넌트와 함께 사용됩니다.

### inkFloat

부유하듯 천천히 움직이는 효과:

```css
@keyframes inkFloat {
  0%, 100% {
    opacity: 0.5;
    transform: translate(0) scale(1);
  }
  50% {
    opacity: 0.7;
    transform: translate(0.625rem, -0.625rem) scale(1.1);
  }
}

.page-decoration--float {
  animation: inkFloat 4s ease-in-out infinite;
}
```

### inkDrift

표류하듯 회전과 함께 움직이는 효과:

```css
@keyframes inkDrift {
  0%, 100% {
    opacity: 0.4;
    transform: translate(0) rotate(0);
  }
  33% {
    opacity: 0.6;
    transform: translate(-0.5rem, 0.5rem) rotate(-5deg);
  }
  66% {
    opacity: 0.5;
    transform: translate(0.375rem, -0.375rem) rotate(3deg);
  }
}

.page-decoration--drift {
  animation: inkDrift 5s ease-in-out infinite;
}
```

### inkSpread

스케일 기반 확산 애니메이션:

```css
@keyframes inkSpread {
  0% {
    opacity: 0.4;
    transform: scale(0);
  }
  100% {
    transform: scale(var(--ink-effect-spread-scale, 2.5));
    opacity: 0;
  }
}
```

## 접근성

<Warning>
애니메이션 접근성 고려사항:

1. **`prefers-reduced-motion` 지원**: 모션 민감 사용자를 위해 애니메이션 비활성화
2. **적절한 duration**: 너무 빠르거나 느린 애니메이션 피하기
3. **무한 애니메이션 주의**: 장식용에만 제한적으로 사용
</Warning>

### 모션 감소 설정 지원

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 개별 컴포넌트에서

```css
.animated-element {
  animation: fadeInUp 0.6s var(--transition-smooth) both;
}

@media (prefers-reduced-motion: reduce) {
  .animated-element {
    animation: none;
    opacity: 1;
  }
}
```
