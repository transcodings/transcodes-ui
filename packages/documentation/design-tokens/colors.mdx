## Brand - 브랜드 색상

Brand 색상은 **Transcodes 브랜드 아이덴티티**를 나타냅니다. 로고, 주요 CTA, 브랜드 강조에 사용됩니다.

### 색상 팔레트

| 토큰 | 값 | 용도 |
|------|-----|------|
| `brand-primary` | `#7c5cff` | 브랜드 메인 색상 |
| `brand-primary-hover` | `#6a4ae8` | 브랜드 호버 상태 |
| `brand-logo-text` | `#1a1a1a` | 로고 텍스트 색상 |

### 사용 예시

```css
/* 브랜드 버튼 */
.brand-button {
  background-color: var(--brand-primary);
  color: white;
}

.brand-button:hover {
  background-color: var(--brand-primary-hover);
}

/* 로고 텍스트 */
.logo-text {
  color: var(--brand-logo-text);
}
```

<Note>
`brand-primary`는 `accent-primary`보다 약간 밝은 보라색입니다. 브랜드 강조가 필요한 곳에 사용하고, 일반 UI 인터랙션에는 `accent-primary`를 사용하세요.
</Note>

## Paper - 배경 색상

Paper 색상은 **배경과 컨테이너**를 위한 중성 색상입니다. 전체 인터페이스의 베이스를 구성합니다.

### 색상 팔레트

| 토큰 | 값 | 용도 |

| `paper-white` | `#faf9fc` | 기본 배경 |
| `paper-cream` | `#f5f4f8` | 보조 배경, 호버 상태 |
| `paper-warm` | `#ebe9f0` | 강조 배경, 액티브 상태 |

## Ink - 텍스트 색상

Ink 색상은 **텍스트 및 타이포그래피**를 위한 색상입니다. 5단계의 명도 레벨을 제공하여 정보 계층을 표현합니다.

### 색상 팔레트

| 토큰 | 값 | 대비율 | 용도 |
|------|-----|--------|------|
| `ink-black` | `#1a1a1a` | **15.8:1** | 주요 텍스트 (제목, 본문) |
| `ink-dark` | `#2d2d2d` | **12.5:1** | 보조 텍스트 (설명, 캡션) |
| `ink-medium` | `#5c5c5c` | **6.0:1** | 약한 강조 텍스트 |
| `ink-light` | `#8a8a8a` | **3.5:1** | 플레이스홀더, 비활성 |
| `ink-faint` | `#c4c4c4` | **1.7:1** | 장식용만 (읽을 수 없음) |

### WCAG 대비율 설명

<Tip>
모든 수치는 `paper-white` 배경(`#faf9fc`) 기준입니다. **WCAG AA 기준(4.5:1)을 초과**하는 토큰만 읽을 수 있는 텍스트에 사용 가능합니다.
</Tip>

<CodeGroup>
  <CodeBlock title="CSS - 텍스트 계층">
    ```css
    /* 주요 제목 */
    h1, h2 {
      color: var(--ink-black);
      /* 대비율: 15.8:1 - 우수 */
    }

    /* 본문 텍스트 */
    p {
      color: var(--ink-black);
      /* 대비율: 15.8:1 - 우수 */
    }

    /* 보조 설명 */
    .caption, .subtitle {
      color: var(--ink-dark);
      /* 대비율: 12.5:1 - 우수 */
    }

    /* 약한 강조 */
    .secondary {
      color: var(--ink-medium);
      /* 대비율: 6.0:1 - 양호 */
    }

    /* 플레이스홀더 (읽을 수 있어야 함) */
    input::placeholder {
      color: var(--ink-light);
      /* 대비율: 3.5:1 - 최소값 */
    }

    /* 장식용 구분선 등 (읽을 수 없음) */
    .divider {
      border-color: var(--ink-faint);
      /* 대비율: 1.7:1 - 장식용만 */
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    // 모든 ink 토큰 접근
    const inkPalette = {
      black: tokens.inkBlack,    // "#1a1a1a"
      dark: tokens.inkDark,      // "#2d2d2d"
      medium: tokens.inkMedium,  // "#5c5c5c"
      light: tokens.inkLight,    // "#8a8a8a"
      faint: tokens.inkFaint,    // "#c4c4c4"
    };
    ```
  </CodeBlock>
</CodeGroup>

### OnAccent 토큰

Accent 색상 배경 위에 텍스트를 표시할 때 사용하는 전용 토큰입니다:

| 토큰 | 라이트 모드 | 다크 모드 | 용도 |
|------|------------|----------|------|
| `alpha-onAccent` | `rgba(255,255,255,0.95)` | `rgba(0,0,0,0.95)` | 주요 텍스트 |
| `alpha-onAccentMuted` | `rgba(255,255,255,0.7)` | `rgba(0,0,0,0.7)` | 보조 텍스트 |

```css
/* Accent 배경 위 텍스트 */
.button-primary {
  background: var(--accent-primary);
  color: var(--alpha-onAccent);
}

.button-primary .subtitle {
  color: var(--alpha-onAccentMuted);
}
```

## Accent - 상호작용 색상

Accent 색상은 **상호작용, 강조, CTA**를 위한 색상입니다. Primary, Success, Warning, Info 4가지 의미론적 상태를 제공합니다.

### 색상 팔레트

#### Primary (기본 강조)

| 토큰 | 값 | 대비율 | 상태 |
|------|-----|--------|------|
| `accent-primary` | `#6b4fd9` | **4.5:1** | 기본 |
| `accent-primary-hover` | `#5a3fc8` | **5.2:1** | 호버 |
| `accent-primary-pressed` | `#4a32b0` | **6.5:1** | 누른 상태 |

#### Success (성공)

| 토큰 | 값 | 대비율 |
|------|-----|--------|
| `accent-success` | `#357a46` | **4.8:1** |
| `accent-success-hover` | `#2d6a3c` | **5.6:1** |
| `accent-success-pressed` | `#255a32` | **6.8:1** |

<CodeGroup>
  <CodeBlock title="CSS - 버튼 상태">
    ```css
    /* 기본 버튼 */
    .button {
      background-color: var(--accent-primary);
      color: white;
    }

    .button:hover {
      background-color: var(--accent-primary-hover);
    }

    .button:active {
      background-color: var(--accent-primary-pressed);
    }

    /* 성공 상태 */
    .success {
      color: var(--accent-success);

      background-color: var(--alpha-success10);
    ```

## Error - 에러 및 경고

Error 색상은 **에러, 경고, 부정적 피드백**을 나타냅니다.

| 토큰 | 값 | 용도 |
|------|-----|------|
| `error-base` | `#c0392b` | 에러 텍스트 |
| `error-bg` | `rgba(231, 76, 60, 0.08)` | 에러 배경 |
| `error-border` | `rgba(231, 76, 60, 0.2)` | 에러 보더 |

```css
.error-message {
  color: var(--error-base);
  background-color: var(--error-bg);
  border: 1px solid var(--error-border);
  border-radius: 4px;
  padding: var(--space-md);
}

input.error {
  border-color: var(--error-base);
}
```

## Semantic - 의미론적 색상

의미론적 색상은 **상태와 피드백**을 전달합니다.

| 토큰 | 값 | 의미 |
|------|-----|------|
| `semantic-warning` | `#b45309` | 경고 |
| `semantic-warning-bg` | `#fff8e1` | 경고 배경 |
| `semantic-success-bg` | `#d1fae5` | 성공 배경 |
| `semantic-info` | `#0369a1` | 정보 |
| `semantic-info-bg` | `#e0f2fe` | 정보 배경 |

## Alpha - 투명도 색상

Alpha 색상은 **반투명 효과**가 적용된 색상으로 호버, 오버레이, 배경 색조에 사용됩니다.

### 주요 Alpha 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `alpha-primary04` | `rgba(124, 92, 255, 0.04)` | 매우 약한 배경 |
| `alpha-primary12` | `rgba(124, 92, 255, 0.12)` | 약한 배경 강조 |
| `alpha-primary20` | `rgba(124, 92, 255, 0.2)` | 보조 호버 |
| `alpha-ink06` | `rgba(26, 26, 26, 0.06)` | 미묘한 구분선 |
| `alpha-ink10` | `rgba(26, 26, 26, 0.1)` | 일반 구분선 |

```css
/* 호버 배경 */
.card:hover {
  background-color: var(--alpha-primary12);
}

/* 구분선 */
.divider {
  border-color: var(--alpha-ink10);
}

/* 오버레이 */
.modal-overlay {
  background-color: var(--alpha-ink10);
}
```

## 색상 사용 가이드

<Warning>
색상만으로 정보를 전달하면 안 됩니다. 색상 맹 사용자를 위해 **텍스트, 아이콘, 패턴** 등으로 정보를 중복 표현하세요.
</Warning>

### 좋은 예

```html
<!-- O: 색상 + 텍스트 -->
<div style="background-color: var(--alpha-success20); color: var(--accent-success);">
  <span style="font-weight: bold;">✓ 성공</span> 데이터가 저장되었습니다.
</div>
```

### 나쁜 예

```html
<!-- X: 색상만 사용 -->
<div style="background-color: var(--alpha-success20);">
  데이터가 저장되었습니다.
</div>
```

## Accessibility 대비율

모든 색상 조합의 **WCAG 대비율**입니다. `paper-white` (#faf9fc) 배경 기준입니다.

### 대비율 기준

| 등급 | 대비율 | 용도 |
|------|--------|------|
| **AAA** | 7:1 이상 | 최고 수준 접근성 |
| **AA** | 4.5:1 이상 | 일반 텍스트 |
| **AA Large** | 3:1 이상 | 큰 텍스트 (18px+ 또는 14px bold) |

### Ink 색상 대비율

| 조합 | 대비율 | WCAG AA | WCAG AAA |
|------|--------|---------|----------|
| ink-black on paper-white | **17.4:1** | Pass | Pass |
| ink-dark on paper-white | **13.8:1** | Pass | Pass |
| ink-medium on paper-white | **6.9:1** | Pass | Fail |
| ink-light on paper-white | **3.7:1** | Large only | Fail |
| ink-faint on paper-white | **2.2:1** | Decorative | N/A |

### Accent 색상 대비율

| 조합 | 대비율 | WCAG AA | WCAG AAA |
|------|--------|---------|----------|
| accent-primary on paper-white | **6.4:1** | Pass | Fail |
| paper-white on accent-primary | **6.4:1** | Pass | Fail |
| accent-success on paper-white | **10.0:1** | Pass | Pass |
| paper-white on accent-success | **10.0:1** | Pass | Pass |
| error-base on paper-white | **6.3:1** | Pass | Fail |

### Semantic 색상 대비율

| 조합 | 대비율 | WCAG AA | WCAG AAA |
|------|--------|---------|----------|
| semantic-warning on paper-white | **10.4:1** | Pass | Pass |
| semantic-info on paper-white | **8.1:1** | Pass | Pass |
