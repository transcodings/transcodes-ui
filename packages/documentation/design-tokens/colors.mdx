## Paper - 배경 색상

Paper 색상은 **배경과 컨테이너**를 위한 중성 색상입니다. 전체 인터페이스의 베이스를 구성합니다.

### 색상 팔레트

| 토큰 | 값 | 용도 |
|------|-----|------|
| `paper-white` | `#faf9fc` | 기본 배경 |
| `paper-cream` | `#f5f4f8` | 보조 배경, 호버 상태 |
| `paper-warm` | `#ebe9f0` | 강조 배경, 액티브 상태 |

<CodeGroup>
  <CodeBlock title="CSS">
    ```css
    /* 기본 배경 */
    body {
      background-color: var(--paper-white);
    }

    /* 보조 섹션 */
    .sidebar {
      background-color: var(--paper-cream);
    }

    /* 카드 호버 */
    .card:hover {
      background-color: var(--paper-warm);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    const bgColor = tokens.paper.white;      // "#faf9fc"
    const secondaryBg = tokens.paper.cream;  // "#f5f4f8"
    const activeBg = tokens.paper.warm;      // "#ebe9f0"
    ```
  </CodeBlock>
</CodeGroup>

## Ink - 텍스트 색상

Ink 색상은 **텍스트 및 타이포그래피**를 위한 색상입니다. 5단계의 명도 레벨을 제공하여 정보 계층을 표현합니다.

### 색상 팔레트

| 토큰 | 값 | 대비율 | 용도 |
|------|-----|--------|------|
| `ink-black` | `#1a1a1a` | **15.8:1** | 주요 텍스트 (제목, 본문) |
| `ink-dark` | `#2d2d2d` | **12.5:1** | 보조 텍스트 (설명, 캡션) |
| `ink-medium` | `#5c5c5c` | **6.0:1** | 약한 강조 텍스트 |
| `ink-light` | `#8a8a8a` | **3.5:1** | 플레이스홀더, 비활성 |
| `ink-faint` | `#c4c4c4` | **1.7:1** | 장식용만 (읽을 수 없음) |

### WCAG 대비율 설명

<Tip>
모든 수치는 `paper-white` 배경(`#faf9fc`) 기준입니다. **WCAG AA 기준(4.5:1)을 초과**하는 토큰만 읽을 수 있는 텍스트에 사용 가능합니다.
</Tip>

<CodeGroup>
  <CodeBlock title="CSS - 텍스트 계층">
    ```css
    /* 주요 제목 */
    h1, h2 {
      color: var(--ink-black);
      /* 대비율: 15.8:1 - 우수 */
    }

    /* 본문 텍스트 */
    p {
      color: var(--ink-black);
      /* 대비율: 15.8:1 - 우수 */
    }

    /* 보조 설명 */
    .caption, .subtitle {
      color: var(--ink-dark);
      /* 대비율: 12.5:1 - 우수 */
    }

    /* 약한 강조 */
    .secondary {
      color: var(--ink-medium);
      /* 대비율: 6.0:1 - 양호 */
    }

    /* 플레이스홀더 (읽을 수 있어야 함) */
    input::placeholder {
      color: var(--ink-light);
      /* 대비율: 3.5:1 - 최소값 */
    }

    /* 장식용 구분선 등 (읽을 수 없음) */
    .divider {
      border-color: var(--ink-faint);
      /* 대비율: 1.7:1 - 장식용만 */
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    // 모든 ink 토큰 접근
    const inkPalette = {
      black: tokens.ink.black,    // "#1a1a1a"
      dark: tokens.ink.dark,      // "#2d2d2d"
      medium: tokens.ink.medium,  // "#5c5c5c"
      light: tokens.ink.light,    // "#8a8a8a"
      faint: tokens.ink.faint,    // "#c4c4c4"
    };
    ```
  </CodeBlock>
</CodeGroup>

## Accent - 상호작용 색상

Accent 색상은 **상호작용, 강조, CTA**를 위한 색상입니다. Primary, Success, Warning, Info 4가지 의미론적 상태를 제공합니다.

### 색상 팔레트

#### Primary (기본 강조)

| 토큰 | 값 | 대비율 | 상태 |
|------|-----|--------|------|
| `accent-primary` | `#6b4fd9` | **4.5:1** | 기본 |
| `accent-primary-hover` | `#5a3fc8` | **5.2:1** | 호버 |
| `accent-primary-pressed` | `#4a32b0` | **6.5:1** | 누른 상태 |

#### Success (성공)

| 토큰 | 값 | 대비율 |
|------|-----|--------|
| `accent-success` | `#357a46` | **4.8:1** |
| `accent-success-hover` | `#2d6a3c` | **5.6:1** |
| `accent-success-pressed` | `#255a32` | **6.8:1** |

<CodeGroup>
  <CodeBlock title="CSS - 버튼 상태">
    ```css
    /* 기본 버튼 */
    .button {
      background-color: var(--accent-primary);
      color: white;
    }

    .button:hover {
      background-color: var(--accent-primary-hover);
    }

    .button:active {
      background-color: var(--accent-primary-pressed);
    }

    /* 성공 상태 */
    .success {
      color: var(--accent-success);
    }

    .success:hover {
      background-color: var(--alpha-success10);
      color: var(--accent-success-hover);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    // Primary 상호작용 색상
    const primaryStates = {
      base: tokens.accent.primary,           // "#6b4fd9"
      hover: tokens.accent['primary-hover'], // "#5a3fc8"
      pressed: tokens.accent['primary-pressed'], // "#4a32b0"
    };

    // Success 색상
    const successStates = {
      base: tokens.accent.success,           // "#357a46"
      hover: tokens.accent['success-hover'], // "#2d6a3c"
      pressed: tokens.accent['success-pressed'], // "#255a32"
    };
    ```
  </CodeBlock>
</CodeGroup>

## Error - 에러 및 경고

Error 색상은 **에러, 경고, 부정적 피드백**을 나타냅니다.

| 토큰 | 값 | 용도 |
|------|-----|------|
| `error-base` | `#c0392b` | 에러 텍스트 |
| `error-bg` | `rgba(231, 76, 60, 0.08)` | 에러 배경 |
| `error-border` | `rgba(231, 76, 60, 0.2)` | 에러 보더 |

```css
.error-message {
  color: var(--error-base);
  background-color: var(--error-bg);
  border: 1px solid var(--error-border);
  border-radius: 4px;
  padding: var(--space-md);
}

input.error {
  border-color: var(--error-base);
}
```

## Semantic - 의미론적 색상

의미론적 색상은 **상태와 피드백**을 전달합니다.

| 토큰 | 값 | 의미 |
|------|-----|------|
| `semantic-warning` | `#b45309` | 경고 |
| `semantic-warning-bg` | `#fff8e1` | 경고 배경 |
| `semantic-success-bg` | `#d1fae5` | 성공 배경 |
| `semantic-info` | `#0369a1` | 정보 |
| `semantic-info-bg` | `#e0f2fe` | 정보 배경 |

## Alpha - 투명도 색상

Alpha 색상은 **반투명 효과**가 적용된 색상으로 호버, 오버레이, 배경 색조에 사용됩니다.

### 주요 Alpha 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `alpha-primary04` | `rgba(124, 92, 255, 0.04)` | 매우 약한 배경 |
| `alpha-primary12` | `rgba(124, 92, 255, 0.12)` | 약한 배경 강조 |
| `alpha-primary20` | `rgba(124, 92, 255, 0.2)` | 보조 호버 |
| `alpha-ink06` | `rgba(26, 26, 26, 0.06)` | 미묘한 구분선 |
| `alpha-ink10` | `rgba(26, 26, 26, 0.1)` | 일반 구분선 |

```css
/* 호버 배경 */
.card:hover {
  background-color: var(--alpha-primary12);
}

/* 구분선 */
.divider {
  border-color: var(--alpha-ink10);
}

/* 오버레이 */
.modal-overlay {
  background-color: var(--alpha-ink10);
}
```

## 색상 사용 가이드

<Warning>
색상만으로 정보를 전달하면 안 됩니다. 색상 맹 사용자를 위해 **텍스트, 아이콘, 패턴** 등으로 정보를 중복 표현하세요.
</Warning>

### 좋은 예

```html
<!-- O: 색상 + 텍스트 -->
<div style="background-color: var(--alpha-success20); color: var(--accent-success);">
  <span style="font-weight: bold;">✓ 성공</span> 데이터가 저장되었습니다.
</div>
```

### 나쁜 예

```html
<!-- X: 색상만 사용 -->
<div style="background-color: var(--alpha-success20);">
  데이터가 저장되었습니다.
</div>
```
