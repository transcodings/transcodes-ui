## 크기 시스템

Transcodes UI의 크기 시스템은 **아이콘**, **스피너**, **화면 장식**, **보더 두께** 등 다양한 크기 토큰을 제공합니다. 일관된 시각적 비율과 반응형 크기 조정을 지원합니다.

## Icon Sizes

아이콘 크기는 **7개 레벨**로 제공됩니다. `action` 사이즈는 버튼 내 아이콘에 최적화되어 있습니다.

### 토큰 목록

| 토큰 | 값 | 픽셀 | 용도 |
|------|-----|------|------|
| `size-icon-xs` | `0.75rem` | 12px | 인라인 아이콘, 배지 |
| `size-icon-sm` | `1rem` | 16px | 작은 버튼, 목록 아이콘 |
| `size-icon-action` | `1.25rem` | 20px | 버튼 내 아이콘, 화살표 |
| `size-icon-md` | `1.5rem` | 24px | 기본 아이콘 크기 |
| `size-icon-lg` | `2rem` | 32px | 강조 아이콘, 네비게이션 |
| `size-icon-xl` | `2.5rem` | 40px | 큰 아이콘, 빈 상태 |
| `size-icon-2xl` | `3rem` | 48px | 히어로 아이콘, 로고 |

### 사용 예시

<CodeGroup>
  <CodeBlock title="CSS">
    ```css
    /* 버튼 내 아이콘 */
    .button-icon {
      width: var(--size-icon-action);
      height: var(--size-icon-action);
    }

    /* 네비게이션 아이콘 */
    .nav-icon {
      width: var(--size-icon-lg);
      height: var(--size-icon-lg);
    }

    /* 빈 상태 일러스트 */
    .empty-state-icon {
      width: var(--size-icon-2xl);
      height: var(--size-icon-2xl);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    const iconSizes = {
      xs: tokens.sizeIconXs,       // "0.75rem"
      sm: tokens.sizeIconSm,       // "1rem"
      action: tokens.sizeIconAction, // "1.25rem"
      md: tokens.sizeIconMd,       // "1.5rem"
      lg: tokens.sizeIconLg,       // "2rem"
      xl: tokens.sizeIconXl,       // "2.5rem"
      '2xl': tokens.sizeIcon2xl,   // "3rem"
    };
    ```
  </CodeBlock>
</CodeGroup>

## Spinner Sizes

로딩 스피너 크기 토큰입니다. `button` 사이즈는 버튼 내 로딩 표시에 최적화되어 있습니다.

### 토큰 목록

| 토큰 | 값 | 픽셀 | 용도 |
|------|-----|------|------|
| `size-spinner-sm` | `1rem` | 16px | 인라인 로딩 |
| `size-spinner-button` | `1.125rem` | 18px | 버튼 내 스피너 |
| `size-spinner-md` | `2rem` | 32px | 카드/섹션 로딩 |
| `size-spinner-lg` | `4rem` | 64px | 페이지 로딩 |
| `size-spinner-xl` | `5rem` | 80px | 전체 화면 로딩 |

### 사용 예시

<CodeGroup>
  <CodeBlock title="CSS">
    ```css
    /* 기본 스피너 */
    .spinner {
      width: var(--size-spinner-md);
      height: var(--size-spinner-md);
      border: 2px solid currentColor;
      border-top-color: transparent;
      border-radius: 50%;
      animation: spin 0.6s linear infinite;
    }

    /* 버튼 내 스피너 */
    .button-spinner {
      width: var(--size-spinner-button);
      height: var(--size-spinner-button);
    }

    /* 페이지 로딩 */
    .page-spinner {
      width: var(--size-spinner-lg);
      height: var(--size-spinner-lg);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    const spinnerSizes = {
      sm: tokens.sizeSpinnerSm,       // "1rem"
      button: tokens.sizeSpinnerButton, // "1.125rem"
      md: tokens.sizeSpinnerMd,       // "2rem"
      lg: tokens.sizeSpinnerLg,       // "4rem"
      xl: tokens.sizeSpinnerXl,       // "5rem"
    };
    ```
  </CodeBlock>
</CodeGroup>

## Screen Icon Sizes

화면 상태 아이콘 크기입니다. 에러/성공/빈 상태 화면에서 사용됩니다. `fluid` 사이즈는 반응형으로 뷰포트에 따라 조정됩니다.

### 토큰 목록

| 토큰 | 값 | 설명 |
|------|-----|------|
| `size-screen-icon-sm` | `4rem` (64px) | 작은 화면 아이콘 |
| `size-screen-icon-md` | `5rem` (80px) | 기본 화면 아이콘 |
| `size-screen-icon-fluid` | `clamp(3rem, 2.43rem + 2.86vw, 5rem)` | 48px → 80px 반응형 |

### 사용 예시

```css
/* 에러/성공 화면 아이콘 */
.screen-icon {
  width: var(--size-screen-icon-fluid);
  height: var(--size-screen-icon-fluid);
}

/* 고정 크기 */
.screen-icon--fixed {
  width: var(--size-screen-icon-md);
  height: var(--size-screen-icon-md);
}
```

## Border Widths

테두리 두께 토큰입니다. 5개 레벨로 시각적 강조 정도를 표현합니다.

### 토큰 목록

| 토큰 | 값 | 픽셀 | 용도 |
|------|-----|------|------|
| `size-border-width-thin` | `0.0625rem` | 1px | 기본 구분선 |
| `size-border-width-base` | `0.09375rem` | 1.5px | 버튼/입력 필드 |
| `size-border-width-thick` | `0.125rem` | 2px | 강조 보더 |
| `size-border-width-heavy` | `0.1875rem` | 3px | 포커스 링 |
| `size-border-width-bold` | `0.25rem` | 4px | 진행 바, 구분자 |

### 사용 예시

<CodeGroup>
  <CodeBlock title="CSS">
    ```css
    /* 기본 구분선 */
    .divider {
      border-top: var(--size-border-width-thin) solid var(--ink-faint);
    }

    /* 버튼 보더 */
    .button-secondary {
      border: var(--size-border-width-base) solid var(--ink-faint);
    }

    /* 포커스 링 */
    .input:focus {
      outline: var(--size-border-width-heavy) solid var(--accent-primary);
      outline-offset: 2px;
    }

    /* 진행 바 */
    .progress-track {
      height: var(--size-border-width-bold);
      background: var(--paper-warm);
    }
    ```
  </CodeBlock>
  <CodeBlock title="TypeScript">
    ```typescript
    import { tokens } from '@transcodes/design-tokens';

    const borderWidths = {
      thin: tokens.sizeBorderWidthThin,   // "0.0625rem"
      base: tokens.sizeBorderWidthBase,   // "0.09375rem"
      thick: tokens.sizeBorderWidthThick, // "0.125rem"
      heavy: tokens.sizeBorderWidthHeavy, // "0.1875rem"
      bold: tokens.sizeBorderWidthBold,   // "0.25rem"
    };
    ```
  </CodeBlock>
</CodeGroup>

## 기타 크기 토큰

### Decoration Blob

페이지 장식 배경 블롭 크기입니다. 반응형으로 뷰포트에 따라 조정됩니다.

| 토큰 | 값 | 설명 |
|------|-----|------|
| `size-decoration-blob` | `clamp(12rem, 9.43rem + 12.86vw, 22rem)` | 192px → 352px 반응형 |

```css
.page-decoration {
  width: var(--size-decoration-blob);
  height: var(--size-decoration-blob);
  border-radius: 50%;
  filter: blur(60px);
}
```

### Close Button

닫기 버튼 크기:

| 토큰 | 값 | 설명 |
|------|-----|------|
| `size-close-button` | `2rem` (32px) | 모달/패널 닫기 버튼 |

```css
.close-button {
  width: var(--size-close-button);
  height: var(--size-close-button);
}
```

### Progress Bar

진행 바 너비:

| 토큰 | 값 | 설명 |
|------|-----|------|
| `size-progress-bar-width` | `12rem` (192px) | 기본 진행 바 너비 |

```css
.progress-bar {
  width: var(--size-progress-bar-width);
}
```

### Chip Padding

칩/태그 패딩:

| 토큰 | 값 | 설명 |
|------|-----|------|
| `size-chip-padding-sm` | `0.125rem 0.5rem` | 작은 칩 패딩 |
| `size-chip-padding-md` | `0.25rem 0.75rem` | 기본 칩 패딩 |

```css
.chip {
  padding: var(--size-chip-padding-md);
  border-radius: var(--radius-full);
}

.chip--sm {
  padding: var(--size-chip-padding-sm);
}
```

## 크기 선택 가이드

<Tip>
**크기 토큰 선택 기준:**

1. **아이콘**: 주변 텍스트 크기와 비례하게 선택
2. **스피너**: 로딩 영역 크기에 맞게 선택
3. **보더**: 시각적 강조 정도에 따라 선택
4. **반응형**: `fluid` 토큰으로 자동 조정
</Tip>

### 아이콘 크기 가이드

| 텍스트 크기 | 권장 아이콘 |
|------------|-----------|
| `font-size-sm` | `size-icon-xs`, `size-icon-sm` |
| `font-size-base` | `size-icon-sm`, `size-icon-action` |
| `font-size-lg` | `size-icon-action`, `size-icon-md` |
| `font-size-xl` | `size-icon-md`, `size-icon-lg` |

### 보더 두께 가이드

| 용도 | 권장 두께 |
|------|----------|
| 구분선, 테이블 | `thin` (1px) |
| 버튼, 입력 필드 | `base` (1.5px) |
| 활성 상태 | `thick` (2px) |
| 포커스 링 | `heavy` (3px) |
| 진행 바, 강조 | `bold` (4px) |
