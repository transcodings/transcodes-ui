---
title: 'Primitives API'
description: '모든 Primitive 컴포넌트의 Props 레퍼런스'
---

모든 컴포넌트는 `sx` prop을 통해 스타일을 커스터마이징할 수 있습니다. 자세한 내용은 [공통 인터페이스](/guides/common-interface)를 참조하세요.

---

## Buttons & Inputs

### tc-button

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'success'` | `'primary'` | 버튼 스타일 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `loading` | `boolean` | `false` | 로딩 상태 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

**Events**: `tc-click`

### tc-input

| `placeholder` | `string` | `''` | 플레이스홀더 |
| `value` | `string` | `''` | 입력값 |
| `error` | `string` | `''` | 에러 메시지 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `required` | `boolean` | `false` | 필수 입력 |
| `name` | `string` | `''` | 폼 필드 이름 |
| `autocomplete` | `string` | `''` | 자동완성 속성 |
| `inputmode` | `string` | `''` | 가상 키보드 모드 |
| `maxlength` | `number` | `undefined` | 최대 길이 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

**Events**: `tc-input`, `tc-change`, `tc-keydown`, `tc-paste`, `tc-blur`

## Text & Display

### tc-text

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `tag` | `'p' \| 'span' \| 'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6' \| 'label' \| 'div'` | `'p'` | HTML 태그 |
| `size` | `'xs' \| 'sm' \| 'base' \| 'lg' \| 'xl' \| '2xl'` | `'base'` | 글자 크기 |
| `weight` | `'400' \| '500' \| '600' \| '700'` | `'400'` | 글자 굵기 |
| `color` | `'primary' \| 'secondary' \| 'tertiary' \| 'success' \| 'error' \| 'warning' \| 'info'` | `'primary'` | 글자 색상 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

<Note>tc-text는 `display: contents`를 사용하므로 margin, padding 등 box model 속성은 적용되지 않습니다. 간격은 tc-section을 사용하세요.</Note>

### tc-icon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `IconName` | required | 아이콘 이름 |
| `sx` | `SxProps` | `{}` | `--icon-size`, `--icon-color` 지원 |

**CSS Variables**:
- `--icon-size`: 기본값 `1.5rem`
- `--icon-color`: 기본값 `currentColor`

### tc-symbol

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `sx` | `SxProps` | `{}` | `--symbol-size`, `--symbol-bg`, `--symbol-color` 지원 |

**CSS Variables**:
- `--symbol-size`: 기본값 `3rem`
- `--symbol-bg`: 기본값 `var(--paper-cream)`
- `--symbol-color`: 기본값 `var(--ink-dark)`

### tc-divider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | `''` | 구분선 중앙 텍스트 |
| `sx` | `SxProps` | `{}` | `--divider-color`, `--divider-spacing` 지원 |

**CSS Variables**:
- `--divider-color`: 기본값 `var(--ink-faint)`
- `--divider-spacing`: 기본값 `var(--space-md)`
