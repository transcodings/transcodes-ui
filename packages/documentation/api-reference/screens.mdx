---

Screen 컴포넌트는 전체 화면을 차지하는 상태 표시 컴포넌트입니다. 로딩, 에러, 성공 상태를 표시할 때 사용합니다.

모든 컴포넌트는 `sx` prop을 통해 스타일을 커스터마이징할 수 있습니다. 자세한 내용은 [공통 인터페이스](/guides/common-interface)를 참조하세요.

---

## tc-loading-screen

로딩 상태를 표시하는 전체 화면 컴포넌트입니다.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `message` | `string` | `''` | 로딩 메시지 |
| `spinnerSize` | `'sm' \| 'md' \| 'lg' \| 'auto'` | `'auto'` | 스피너 크기 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

```html
<tc-loading-screen message="데이터를 불러오는 중..."></tc-loading-screen>
```

---

## tc-error-screen

에러 상태를 표시하는 전체 화면 컴포넌트입니다.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `'오류가 발생했습니다'` | 에러 제목 |
| `message` | `string` | `''` | 에러 상세 메시지 |
| `retryLabel` | `string` | `'다시 시도'` | 재시도 버튼 텍스트 |
| `showRetry` | `boolean` | `true` | 재시도 버튼 표시 여부 |
| `autoAnimate` | `boolean` | `true` | 애니메이션 활성화 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

**Events**: `tc-retry`

```html
<tc-error-screen
  title="연결 오류"
  message="서버에 연결할 수 없습니다."
  retryLabel="다시 연결"
  @tc-retry=${handleRetry}
></tc-error-screen>
```

## tc-success-screen

성공 상태를 표시하는 전체 화면 컴포넌트입니다.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `'완료'` | 성공 제목 |
| `message` | `string` | `''` | 성공 상세 메시지 |
| `actionLabel` | `string` | `''` | 액션 버튼 텍스트 (비어있으면 버튼 숨김) |
| `autoAnimate` | `boolean` | `true` | 애니메이션 활성화 |
| `sx` | `SxProps` | `{}` | 스타일 오버라이드 |

**Events**: `tc-action`

```html
<tc-success-screen
  title="결제 완료"
  message="주문이 성공적으로 처리되었습니다."
  actionLabel="주문 내역 보기"
  @tc-action=${handleAction}
></tc-success-screen>
```

## 사용 패턴

### 상태에 따른 화면 전환

```typescript
import '@transcodes/ui-components/screens';

type ScreenState = 'loading' | 'error' | 'success' | 'content';

function renderScreen(state: ScreenState) {
  switch (state) {
    case 'loading':
      return html`<tc-loading-screen message="처리 중..."></tc-loading-screen>`;
    case 'error':
      return html`<tc-error-screen @tc-retry=${retry}></tc-error-screen>`;
    case 'success':
      return html`<tc-success-screen title="완료"></tc-success-screen>`;
    default:
      return html`<!-- 메인 콘텐츠 -->`;
  }
}
```

### React에서 사용

```tsx
import '@transcodes/ui-components/screens';

function App() {
  const [state, setState] = useState<'loading' | 'error' | 'success'>('loading');

  if (state === 'loading') {
    return <tc-loading-screen message="로딩 중..." />;
  }

  if (state === 'error') {
    return (
      <tc-error-screen
        title="오류"
        message="문제가 발생했습니다."
        onTcRetry={() => setState('loading')}
      />
    );
  }

  return <tc-success-screen title="완료!" />;
}
```
